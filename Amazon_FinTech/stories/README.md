## Story template

Copy into a new file when drafting:

```markdown
# [LP]: [Short title]

**Leadership Principle:** [exact LP name]

## STAR

**Situation:** [1–2 sentences — context, stakes]

**Task:** [What you personally owned]

**Action:** [What *I* did — steps, decisions, data used]

**Result:** [Quantified outcome + business impact]

**Lesson:** [What I'd do differently or apply next time]

## Interview notes

- Likely follow-ups:
- Weak spots to tighten:
- Say "I" highlights:
```

## Refinement loop

After `/debrief` or mock feedback:

1. Note which story felt weak
2. Edit the story file — add metric, sharpen "I", shorten to 90 sec
3. If the same correction repeats twice, add a line to root [`AGENTS.md`](../../AGENTS.md)
